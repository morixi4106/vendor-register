# 出店者基本契約書（原案）

> **INTERNAL DRAFT / 未施行 / 法的効力なし** 出店者との現行契約ではありません。

> 公開前に事業者情報、料金、精算条件、損害賠償上限及び管轄裁判所を確定してください。本原案は現行システムの挙動を契約へ反映するための作業文書です。

## 第1条（目的）

本契約は、[運営者正式名称]（以下「運営者」といいます。）が運営するオンラインマーケットプレイス「Oja Immanuel Bacchus」（以下「本サービス」といいます。）へ商品を出品する事業者（以下「出店者」といいます。）との間で、本サービスの利用、商品の販売、代金受領、精算、返品及び責任分担について定めるものです。

## 第2条（契約上の地位）

1. 出店者は、出店者が掲載する商品の購入者に対する売主となります。
2. 運営者は、運営者が直接販売する商品を除き、出店者商品の売主ではありません。
3. 運営者は、出店者から委託を受け、注文受付、購入者窓口、代金受領、返金処理及び精算事務を行います。
4. 出店者は、購入者が運営者又は運営者が指定する決済サービスへ代金を支払った時点で、当該購入者の出店者に対する支払義務が履行されたものとして扱うことに同意します。

## 第3条（登録及び本人確認）

1. 出店者は、名称、住所、代表者、電話番号、メールアドレス、振込口座、法人番号又は本人確認情報その他運営者が求める情報を正確に登録します。
2. 出店者は、登録情報に変更が生じた場合、遅滞なく更新します。
3. 運営者は、本人確認、事業実態、口座名義、反社会的勢力との関係、許認可及び商品安全に関する資料を求めることができます。
4. 出店者は、自らが反社会的勢力に該当せず、これらと取引又は関係を有しないことを表明し、保証します。

## 第4条（商品情報及び法令遵守）

1. 出店者は、商品名、画像、説明、価格、在庫、状態、原産国、重量、配送条件及び注意事項を正確に登録します。
2. 出店者は、出品商品が真正なものであり、第三者の知的財産権を侵害せず、安全性、品質、表示及び広告に関する法令へ適合することを保証します。
3. 出店者は、PSE、PSCその他法令上必要なマーク、届出、検査、許認可及び警告表示を確認し、求められた場合は証拠を提出します。
4. 出店者は、リコール、事故、行政照会、権利侵害の申立てその他商品の販売継続に影響する事情を知った場合、直ちに運営者へ通知します。

## 第5条（掲載審査及び停止）

1. 運営者は、商品の適法性、安全性、真正性又は表示の正確性を確認するため、資料提出、修正又は追加説明を求めることができます。
2. 運営者は、違法、危険、権利侵害、虚偽表示又は購入者被害のおそれがある商品を、事前通知なく一時停止できます。
3. 掲載審査は、出店者の責任を免除又は運営者へ移転するものではありません。

## 第6条（注文及び売買契約）

1. 出店者商品の売買契約は、購入者の注文を本サービスが受け付け、決済が承認された時点で、出店者と購入者との間に成立します。ただし、在庫不存在、明白な価格誤表示又は法令上販売できない事情がある場合の取扱いは、購入者規約に従います。
2. 一つの注文に複数出店者の商品が含まれる場合、売買契約は出店者ごとに成立します。
3. 出店者は、管理画面に表示された注文について、定められた期限までに発送又は対応不能の連絡を行います。

## 第7条（配送及び返品受取先）

1. 出店者は、登録した発送元から適切に梱包して発送し、追跡番号等を登録します。
2. 出店者は、返品商品を実際に受領できる返送先を、一般住所とは別に明示的に登録・確認します。
3. 出店者は、購入者へ案内済みの返送先で返品を受領します。案内後の住所変更は、当該案内済み返送先を変更しません。
4. 複数店舗注文では、購入者は店舗ごとに別送します。出店者は他店舗商品の受領を強制されません。

## 第8条（返品、撤回、キャンセル及び返金）

1. 出店者は、適用法、購入者規約及び商品ページに表示された条件に従い、返品、撤回、キャンセル及び返金へ対応します。
2. EUの撤回権が適用される場合、通常配送方法に相当する初回送料は注文全体で一度だけ返金対象として確認し、返送費用は適法な事前表示がある範囲で購入者負担とします。
3. 商品の価値が必要な確認を超える取扱いにより減少した場合、証拠を確認したうえで適法な範囲の減額を判断します。
4. 運営者は購入者の申請受信のみを理由に自動返金又は自動キャンセルを行いません。
5. 出店者は、返品到着、商品状態及び意見を所定の期限までに記録します。

## 第9条（手数料及び精算）

1. 運営者は、商品代金その他購入者から受領した金額から、次の金額を控除した残額を出店者へ精算します。
   - プラットフォーム手数料: [要確定]
   - 決済手数料の負担分: [要確定]
   - 出店者負担と確定した返金、チャージバック、再配送及び損害額
   - 税金、振込手数料その他本契約で明示した金額
2. 精算周期は[要確定]、最低精算額は[要確定]円、通常の保留期間は[要確定]日とします。
3. 出店者は、自らに課される税金、申告及びインボイス対応について責任を負います。
4. 運営者は、台帳及び精算明細を電磁的方法で提供します。

## 第10条（保留、準備金及び負残高）

1. 運営者は、返金、チャージバック、商品事故、権利侵害、行政調査、詐欺又は出店者の支払能力に合理的な懸念がある場合、必要な範囲で出金を保留できます。
2. 運営者は、想定される請求額及びリスクに相当する範囲で準備金を設定できます。
3. 保留又は準備金を設定した場合、法令又は調査上支障がない限り、理由、金額及び見直し時期を出店者へ通知します。
4. 保留又は準備金は、出店者の責任確定そのものを意味しません。

## 第11条（責任確定及び相殺）

1. 運営者は、出店者へ負担を求める前に、注文、商品、配送、連絡履歴その他の証拠を確認し、責任の有無及び金額を記録します。
2. 出店者の責任額が確定した場合、運営者は、出店者へ支払うべき確定済み又は将来の売上金と当該責任額を対当額で相殺できます。
3. 相殺額は、当該案件について確定した出店者責任額を超えません。
4. 相殺後も不足する場合、運営者は出店者へ直接請求できます。出店者は請求日から[要確定]日以内に支払います。
5. 責任が確定していない案件については、相殺を行わず、第10条の保留又は準備金により対応します。

## 第12条（購入者情報）

1. 出店者は、購入者情報を注文履行、配送、返品、法令対応及び問い合わせ対応の目的に限って使用します。
2. 出店者は、購入者情報を広告、名簿作成、第三者提供又は本サービス外の勧誘へ使用しません。
3. 出店者は、適切な安全管理措置を講じ、漏えい等を知った場合は直ちに運営者へ連絡します。
4. 取引終了後、法令上の保存義務を除き、不要となった購入者情報を安全に削除します。

## 第13条（苦情、事故及び行政対応）

1. 出店者は、商品事故、重大な苦情、行政機関又は権利者からの連絡を受けた場合、直ちに運営者へ報告します。
2. 出店者は、調査、回収、購入者通知、返金、証拠保全及び行政報告へ協力します。
3. 出店者の商品に起因する費用は、出店者の責任が確定した範囲で出店者が負担します。

## 第14条（保険）

運営者は、商品区分又は販売規模に応じて、出店者にPL保険その他適切な保険への加入及び証券の提出を求めることができます。

## 第15条（損害賠償及び補償）

1. 各当事者は、自らの契約違反、故意又は過失により相手方へ生じた通常かつ直接の損害を賠償します。
2. 出店者は、商品の欠陥、違法性、表示違反又は権利侵害により運営者が第三者へ支払った合理的な費用を、出店者の責任が確定した範囲で補償します。
3. 損害賠償の上限は[要確定]とします。ただし、故意又は重過失、個人情報漏えい、知的財産権侵害、商品事故及び未払金には上限を適用しないものとするか、別途上限を定めます。

## 第16条（利用停止及び契約終了）

1. 運営者は、重大な契約違反、虚偽登録、危険商品、権利侵害、支払不能又は反社会的勢力との関係が判明した場合、出品又はアカウントを停止できます。
2. 契約終了後も、既存注文、返品、返金、チャージバック、個人情報、未払金及び責任に関する条項は存続します。

## 第17条（契約変更）

1. 運営者は、法令変更、サービス変更その他合理的な必要がある場合、本契約を変更できます。
2. 重要な変更は、効力発生日までに電磁的方法で通知し、必要に応じて出店者から改めて同意を取得します。
3. 契約版、本文ハッシュ、同意者及び同意日時を保存します。

## 第18条（準拠法及び管轄）

本契約は日本法を準拠法とし、本契約に関する第一審の専属的合意管轄裁判所を[要確定]地方裁判所又は簡易裁判所とします。
