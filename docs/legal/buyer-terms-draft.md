# 購入者向け利用規約（原案）

> 特定商取引法表示、返品ポリシー、プライバシーポリシー及びEU向け撤回案内と併せて公開することを前提としています。

## 1. 適用

本規約は、オンラインマーケットプレイス「Oja Immanuel Bacchus」で商品を購入するお客様と、マーケットプレイス運営者及び各商品の販売店舗との関係に適用されます。

## 2. 商品の売主

1. 各商品ページ及び注文確認に表示される販売店舗が、その商品の売主です。
2. 「Oja Immanuel Bacchus」が販売する運営直販商品については、運営者が売主です。
3. 一つの注文に複数店舗の商品が含まれる場合、お客様と各販売店舗との間に、店舗ごとの売買契約が成立します。
4. 運営者は、出店店舗商品の売主ではありませんが、注文、決済、問い合わせ、返品及び返金の窓口を提供します。

## 3. 注文及び契約成立

1. お客様が注文を送信し、決済が承認され、注文確認が表示された時点で売買契約が成立します。
2. 在庫不存在、明白な価格誤表示、法令上の販売禁止、不正利用その他やむを得ない事情がある場合、運営者又は売主は注文を取り消すことがあります。この場合、受領済み代金を返金します。
3. 注文確認画面及び確認メールは保存してください。

## 4. 価格及び支払い

1. 商品価格、税、送料その他の費用は、注文確定前に表示します。
2. お客様は、Shopify Checkoutに表示される決済方法で支払います。
3. 運営者は、出店店舗から授権を受け、出店店舗商品の代金を代理受領します。お客様が運営者又は指定決済サービスへ支払った時点で、当該売主への支払いも完了します。

## 5. 配送

1. 商品は、商品を販売する店舗又は指定された発送拠点から発送されます。
2. 複数店舗の商品は、別々の荷物、追跡番号又は時期で届く場合があります。
3. 配送予定日は目安であり、天候、通関、配送会社その他合理的に管理できない事情により変更される場合があります。
4. 配送不能又は住所不備により追加費用が生じた場合の取扱いは、注文時の配送条件に従います。

## 6. 検品及び連絡

商品到着後、商品、数量及び状態を確認してください。誤配送、破損、欠品又は説明との重大な相違がある場合、注文番号、写真その他確認に必要な情報を添えて、合理的な期間内に連絡してください。

## 7. キャンセル、返品及び返金

1. 発送前のキャンセル可否は、注文状況及び商品特性により判断します。
2. 商品不良、誤配送その他売主の責任による返品費用は、売主又は運営者が案内する方法で負担します。
3. お客様都合の返品は、商品ページ及び返品ポリシーに表示された条件に従います。
4. 複数店舗注文の返品は、運営者の案内に従い、店舗ごとに別の返送先へ発送してください。異なる店舗の商品を一つの荷物へまとめないでください。
5. 返金は、返品到着、商品状態、決済状況及び適用法を確認した後、原則として元の決済方法へ行います。

## 8. EUのお客様の撤回権

1. EUの消費者保護法が適用される場合、お客様は、適用除外商品を除き、商品受領後14日以内に理由を示さず撤回を通知できる場合があります。
2. 撤回が認められる場合、商品代金及び最も安価な通常配送方法に相当する初回送料を返金対象として確認します。通常配送より高額な配送方法を選択したことによる追加費用は、返金対象外となる場合があります。
3. 返送費用は、売主が負担すると案内した場合又は法令により売主負担となる場合を除き、お客様負担となる場合があります。
4. 商品の性質、特徴及び機能を確認するために必要な範囲を超えて取り扱い、価値を減少させた場合、その減少分を返金額から控除することがあります。
5. カスタム商品、傷みやすい商品、開封された衛生商品その他法令上の適用除外となる商品には、撤回権が適用されない場合があります。
6. 撤回通知を受け付けただけで、自動的にキャンセル又は返金が確定するものではありません。申請日時は固定し、注文、返送及び商品状態を確認して処理します。

## 9. 商品安全及びリコール

商品に安全上の問題があると判明した場合、運営者又は売主は、販売停止、購入者への連絡、回収、交換又は返金を行うことがあります。購入者は、案内に従って商品の使用を停止し、必要な情報提供に協力してください。

## 10. 禁止行為

不正決済、虚偽の返品、他人の情報の使用、サービスへの攻撃、転売規制の回避その他違法又は不正な行為を禁止します。

運営者及び売主は、注文履行、配送、返品、返金、問い合わせ、不正防止及び法令対応のため、必要な範囲で購入者情報を取り扱います。詳細はプライバシーポリシーに定めます。

1. 各売主は、自らが販売する商品の契約適合性、安全性、表示及び法令遵守について責任を負います。
2. 運営者は、運営直販商品及び運営者自身のプラットフォーム運用について責任を負います。
3. 消費者契約法その他の強行法規に反して、運営者又は売主の責任を免除するものではありません。
