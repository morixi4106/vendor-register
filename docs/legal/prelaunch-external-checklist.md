# アプリ外・公開前チェックリスト

> **INTERNAL DRAFT** チェック済みであること自体は専門家・決済会社の承認を意味しません。証跡は非公開領域へ保存してください。

## 判定ルール

- `BLOCKER`: 国内公開前に完了が必要
- `EU BLOCKER`: 国内限定では保留できるが、EU販売前に必要
- `POST-LAUNCH`: 公開後の期限を決めて対応可能

チェック欄には、単なる「確認済み」ではなく、契約PDF、メール回答、画面、保険証券、議事録等の証跡URLを記録します。

## 9. 専門家への限定照会

外部専門家へ丸投げせず、次の質問だけを書面で確認します。

### 弁護士又は決済規制担当

- [ ] 現行の代金代理受領と月次精算について、必要な登録・届出の有無
- [ ] 出店者契約の相殺、準備金、損害賠償及び責任分担の有効性
- [ ] 特商法及び取引DPF消費者保護法上の表示・開示手順

### 税理士

- [ ] 代理受領額、店舗売上、運営手数料及び直販売上の仕訳
- [ ] 領収書・インボイスの発行主体と複数店舗注文の表示
- [ ] 返金、送料、チャージバック、相殺及び越境送金の税務処理

### 保険代理店

- [ ] マーケットプレイス運営及び直販に必要なPL・サイバー補償
- [ ] 出店店舗へ要求すべき保険条件

## 10. 最終GO条件

国内公開は、全 `BLOCKER` に証跡があり、アプリ側の本番確認と少額実取引が成功した場合に限りGOとします。EU販売は、国内GOに加え、全 `EU BLOCKER` 完了後に国単位で開放します。
