# 専門家・決済会社への限定照会テンプレート

## 1. 決済会社への照会

件名: マーケットプレイスにおける第三者販売代金の代理受領可否について

```text
当社は、複数の出店店舗が商品を販売するオンラインマーケットプレイスを運営します。

・出店店舗の商品について、購入者に対する売主は各出店店舗です。
・当社直販商品についてのみ、当社が売主です。
・購入者はShopify Checkoutで一括決済します。
・決済加盟店及び入金口座の名義は当社です。
・出店店舗は当社へ商品代金の代理受領権限を付与します。
・当社は注文を店舗別に台帳管理し、返金等を反映した後、月次で店舗へ精算します。
・購入者や店舗間の送金、残高譲渡、現金チャージその他の汎用送金機能はありません。
・店舗への精算は管理者承認後に行います。

上記の取引が、貴社の加盟店規約及び利用可能な事業類型に適合するか、書面で回答をお願いします。
また、追加契約、Marketplace申請、サブマーチャント登録、KYC又は禁止事項があればご教示ください。
```

## 2. 金融規制を扱う弁護士への照会

件名: 商品販売マーケットプレイスの代金代理受領・店舗精算に関する登録要否

添付:

- `marketplace-legal-structure.md`
- `funds-flow-tax-memo.md`
- 出店者基本契約原案
- 決済会社からの回答
- 実際の精算画面及び口座名義が分かる資料

質問:

```text
1. 添付の資金フローは、資金決済法上の資金移動業その他の登録・届出を要しますか。
2. 収納代行又は代金代理受領として運用するために、契約・画面・資金管理上追加すべき条件はありますか。
3. 精算周期、保留期間、準備金、将来売上との相殺及び直接請求に制約はありますか。
4. 越境出店者への送金を開始する際に、再評価が必要な論点は何ですか。
5. 取引量又は残高が一定規模を超えた場合に必要となる対応はありますか。

一般論ではなく、添付フローを前提とした結論、前提条件及び見直し条件を書面でお願いします。
```

## 3. EC・消費者法を扱う弁護士への照会

```text
1. 各出店店舗を商品ごとの売主とし、運営が窓口と代金受領を行う表示・規約は整合していますか。
2. 特定商取引法に基づく表示は、店舗ページ、商品ページ、注文確認のどこに何を出す必要がありますか。
3. 取引デジタルプラットフォーム消費者保護法に基づく連絡、調査、本人確認及び情報開示手順は十分ですか。
4. 出店者基本契約の保留、準備金、相殺、直接請求及び損害賠償条項に修正が必要ですか。
5. 購入者規約の返品、商品事故、複数店舗注文及び運営の責任表示に修正が必要ですか。
6. EU販売開始前に、国別で追加確認すべき強行法規を列挙してください。
```

## 4. 税理士への照会

```text
添付資金フローを前提に、次を仕訳例付きで回答してください。

1. Shopifyから運営口座へ一括入金された、出店店舗商品の代金。
2. 運営直販商品の代金。
3. 運営のプラットフォーム手数料と決済手数料。
4. 店舗別の商品返金、初回送料返金、チャージバック。
5. 責任確定前の準備金と、確定後の相殺・直接請求。
6. 免税店舗とインボイス登録店舗が混在する注文。
7. 購入者へ交付する領収書・適格請求書と、店舗へ交付する精算明細。
8. 証憑の保存期間と、月次照合資料。
```

## 5. 保険代理店への照会

```text
当社は、自社直販と第三者出店店舗の商品を併売するオンラインマーケットプレイスです。

次の事故を対象に、必要な保険種類、支払限度、免責及び出店店舗へ要求すべき保険条件をご提案ください。

・直販商品の製造物責任
・出店店舗商品の事故について運営者も請求を受ける場合
・商品回収、購入者通知及び緊急配送費用
・個人情報漏えい、ランサムウェア及び事業停止
・知的財産権侵害の申立て
・海外購入者又は越境配送に関する請求
```
